bootcamps

Nos Bootcamps sont des ateliers de 4 jours ouverts à tou.te.s.
Grâce à une immersion dans le Makers' lab, les participants y apprennent les bases d'une technologie par la pratique et découvrent les enjeux et problématiques qu'elle soulève dans la création d'un projet collectif opensource.
Chaque midi, un intervenant anime une conférence inspirante suivie d'une session de questions-réponses permettant d'explorer une problématique soulevée par la création du projet.
Le dernier jour, les participants collaborent en équipe de 4 à 6 personnes pour fabriquer et documenter un projet impactant.

➔ Ne manquez pas nos prochains bootcamps!

bootcamp fab

Apprenez l'électronique pour piloter des moteurs, la programmation pour communiquer avec les machines, la conception 3d pour fabriquer une imprimante 3d et la documentation pour partager les sources de votre travail.

bootcamp code

Apprenez l'électronique pour construire des manettes de jeu, le dessin vectoriel et le maniement d'une découpeuse laser pour fabriquer la table d'arcade, la programmation pour créer des jeux vidéo rétro et et la documentation pour partager les sources de votre travail.

bootcamp IdO

Apprenez l'électronique pour collecter des données à partir de capteurs, le dessin vectoriel et le maniement d'une découpeuse laser pour fabriquer une ruche, le développement web pour créer un tableau de bord en ligne et la documentation pour partager les sources de votre travail.

bootcamp robotique

Apprenez l'électronique pour piloter des moteurs, le dessin vectoriel et le maniement d'une découpeuse laser pour fabriquer un robot, le développement web pour créer un espace de visio-conférence et la documentation pour partager les sources de votre travail.

design sprints

Nos design sprints sont des ateliers de 4 jours avec une sélection sur un appel à projets.
Bénéficiez d'expertises dédiées pour aider votre équipe à apprendre l'agilité et à développer votre propre prototype. Testez votre prototype avec des utilisateurs le dernier jour pour pouvoir améliorer votre projet.
Vous avez un projet d'entreprise? Réalisez-le!

➔ Ne manquez pas nos prochains design sprints!
